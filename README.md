## Requirements

All dependencies listed in package.json

- Node v8.10.0
- npm v3.5.2
- MongoDB v3.6.3

### Installing Dependencies

From within the root directory:

```sh
npm install
```

## Development

From within the root directory:

To run server

```sh
npm start
```

To run dev environment/webpack

```sh
npm run react-dev
```

To run tests

```sh
npm test
```
